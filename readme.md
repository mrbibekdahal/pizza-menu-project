#  Python Pizza Ordering System

## Overview

This is a GUI-based pizza ordering system built using Python and Tkinter. The application allows users to browse pizzas, view details, add items to a cart, and calculate the total price.

# Features

* Display pizzas with images
* Load data from CSV file
* Add to cart functionality
* Quantity selection
* Order total calculation
* Simple and interactive GUI

# Technologies Used

* Python
* Tkinter (GUI)
* CSV (data handling)
* PIL (image processing)

## ▶️ How to Run

1. Install required library:
   pip install pillow

2. Run the program:
pizza menu.py
